# Adult Census Income Prediction

A Machine Learning project built using Python to predict whether a person's income exceeds $50K annually using demographic and employment-related data.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SMOTE
- Jupyter Notebook

## Project Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Feature Encoding
- Handling Missing Values
- Handling Imbalanced Data using SMOTE
- Machine Learning Model Training
- Model Evaluation and Prediction

## Dataset Information
The dataset includes attributes such as:
- Age
- Education
- Occupation
- Workclass
- Marital Status
- Hours per Week
- Capital Gain/Loss
- Native Country

Target:
- Income Classification (`<=50K` or `>50K`)

## Machine Learning Process
1. Data Preprocessing
2. Feature Engineering
3. Data Visualization
4. Train-Test Split
5. SMOTE Oversampling
6. Model Training
7. Evaluation
8. Prediction
